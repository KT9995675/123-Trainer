## 1. Суть проекта

Онлайн-тренажёр по математике для школьников:
- **Google Apps Script** + **Google Таблица** (БД) + **Google Диск** (картинки условий)
- Ученик в браузере: вход → задачи случайной ступени → 7 верных / макс. 3 ошибки → ступень
- 5 ступеней, старт с 0 (решает задачи ступени 1)

## 4. Схема таблицы «Пользователи» (одна строка = ученик)

| Колонка | Поле |
|---|---|
| A | ID |
| B | Имя |
| C | Пароль |
| D | Телефон |
| E | Класс |
| F | Город |
| G | Школа |
| H | Текущая ступень (0–5) |
| I | Верных в сессии |
| J | Ошибок в сессии |
| K… | Лог сессий (см. ниже) |

### Формат лога (с v0.4)

После каждого маркера сессии `##`:

```
##  |  101,102,103,...  |  101  |  Да  |  102  |  Нет  | ...
 ↑         ↑ очередь 10 ID      ↑ пары (ID + вердикт одновременно, с v0.5)
маркер
```

- Лог **не стирается** при win/fail — только I,J обнуляются + новый `##`
- Счётчики сессии обнуляются при: входе, win, fail

## 5. Лист «Задачи»

| A | B | C | D |
|---|---|---|---|
| Номер ID | Ссылка на картинку (Drive) | Подсказка | Ответ |

ID: первая цифра = ступень (`101`, `102` → ступень 1).

---

## 6. Правила игры (актуальные)

- **Победа:** 7 **верных** ответов в сессии (`верно >= 7`)
- **Провал:** 3 **неверных** (`ошибки >= 3`)
- Задачи в сессии **не повторяются** (очередь 10 ID)
- Подсказка — до отправки ответа
- Ответы **не защищаем** — отдаём на клиент в `sessionTasks` (с v0.5)
- Новая сессия: при **входе** и после **win/fail**

## 7. Архитектура кода (v0.6)

### Сервер (`Code.gs`) — ключевые функции

| Функция | Назначение |
|---|---|
| `doGet` | Отдаёт Index.html |
| `loginUser` | Вход; возвращает `{ id, name, rowNum }` |
| `getTaskForUser(userId, startNewSession)` | Новая сессия: 10 ID + кэш; отдаёт `sessionTasks` (с answer), `sessionIndex` |
| `submitAnswer(userId, taskId, isCorrect, rowNum, sessionIndex)` | **Только запись:** пара `(ID, Да/Нет)` + H,I,J; использует CacheService |
| `_createSessionQueue` | 10 случайных задач → CSV в таблицу + `sessionTasks` |
| `_putSessionCache` / `_getSessionCache` | Кэш сессии на сервере (6 ч) |
| `_parseCurrentSessionLog` | Fallback: хвост строки 32 ячейки |
| `_getTasksByLevel` | Задачи по первой цифре ID |
| `_resetSessionCounters` | I,J=0 + маркер `##` |

### Клиент (`Index.html`) — ключевые функции

| Функция | Назначение |
|---|---|
| `handleLogin` | Вход → `loadNextTask` с `startNewSession` |
| `loadNextTask` | Сервер **только** при новой сессии (вход / win / fail / «Продолжить») |
| `handleSendAnswer` | Проверка **на клиенте** (`checkAnswerLocally`); сервер — запись; кнопка «Сохраняем...» |
| `handleNextStep` | `continue` → `showTaskFromSession` **без сервера** |
| `showTaskFromSession` | Показ из `sessionTasks[]` |
| `prefetchNextImage` | Prefetch картинки следующей задачи |
| `renderApples` / `renderHearts` | 7 яблочек + 3 сердца |

### UI экрана тренажёра

- Имя + ступень (решаемая = `studentLevel + 1`)
- 7 яблочек (верные), 3 сердца (ошибки)
- Картинка, подсказка, поле ответа
- «Верно!» / «Ошибка! Ответ: …» + «Дальше»

**Не используется (мертвый код):** `handleAnswerResponse` — оставлен в файле, не вызывается.
